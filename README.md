# Keyboard Layout Info

Display current active layout and list of installed layouts info:

- Layout name (English)
- HKL
- KLID
- Locale (from HKL)
- Locale (from KLID)
- Country code (from HKL)
- Country code (from KLID)
- Country name (from HKL)
- Country name (from KLID)

## Requirements

```
pip install -r requirements.txt
```

## Run

```
python -m keyboard_layout_info
```

## Build exe

```
python scripts/build.py
```
